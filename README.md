# НефтехимОбучение — обучающая система по оборудованию резервуарных парков

Дипломный проект (специальность «Информационные системы и программирование»).
Обучающий веб-сайт по технологическому оборудованию для резервуарных парков АЗС,
нефтебаз и НПЗ на основе номенклатуры ЗАО «СПТО Нефтехимпроект».

## Возможности
- **Каталог-учебник** — карточки оборудования: назначение, принцип работы,
  характеристики, применение, нормативка, требования безопасности.
- **Поиск и фильтры** по названию, марке и категориям.
- **Интерактивная схема резервуара РВС** (SVG) с горячими точками, ведущими в каталог.
- **Личный кабинет** — избранное и отметки «изучено», прогресс изучения.
- **Админ-панель** — CRUD категорий и оборудования без правки кода.

## Стек
Python · Flask · SQLAlchemy · Flask-Login · SQLite (локально) / PostgreSQL (Render) ·
HTML/CSS/JS · SVG · gunicorn.

## Запуск локально (Windows / PowerShell)
```powershell
python -m venv .venv
.\.venv\Scripts\Activate.ps1
pip install -r requirements.txt
python seed.py          # создаёт БД и наполняет контентом
python wsgi.py          # запуск, http://127.0.0.1:5000
```

Администратор по умолчанию: `admin` / `admin123`
(меняется переменными окружения `ADMIN_LOGIN`, `ADMIN_PASSWORD`).

## Развёртывание на Render
Репозиторий содержит `render.yaml` (Blueprint): создаёт веб-сервис и бесплатную
базу PostgreSQL, прогоняет `seed.py` перед стартом.

1. Залить проект в репозиторий GitHub.
2. На Render → **New → Blueprint**, указать репозиторий.
3. Задать `ADMIN_LOGIN` и `ADMIN_PASSWORD` в настройках сервиса.
4. Deploy.

Переменные окружения:
| Переменная | Назначение |
|------------|-----------|
| `DATABASE_URL` | строка подключения PostgreSQL (подставляется Render автоматически) |
| `SECRET_KEY` | ключ сессий (генерируется Render) |
| `ADMIN_LOGIN` / `ADMIN_PASSWORD` | учётные данные администратора |
